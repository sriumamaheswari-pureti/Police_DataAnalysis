
## 📌 Project Title

**🚓 Police Data Analysis using Python**

---

## 📖 Project Description

This project involves analyzing a real-world police dataset to understand traffic stop patterns and driver behavior. Using Python (Pandas), the dataset was cleaned, processed, and explored to extract meaningful insights related to gender, violations, stop duration, and age distribution.

---

## 🎯 Objectives

* Clean and preprocess raw data
* Handle missing values effectively
* Analyze traffic stop patterns
* Identify trends based on gender and violations
* Derive meaningful insights

---

## 📂 Dataset

* Police Data (CSV format)
* Includes Gender, Age, Violation, Stop Duration, Search Conducted

---

## 🔄 Project Workflow

```text
Data Collection → Data Loading → Data Cleaning → Data Exploration → Data Transformation → Analysis → Insights
```

### 🔹 Workflow Breakdown

* **Data Loading:** Imported dataset using Pandas
* **Data Cleaning:** Removed null columns and handled missing values
* **Data Exploration:** Used functions like head(), value_counts() to understand data
* **Data Transformation:** Applied mapping and datatype conversions
* **Analysis:** Used filtering, groupby, and aggregation techniques
* **Insights:** Derived patterns related to gender, violations, and stop duration

---

## 🔍 Analysis Performed

* Removed columns with only missing values
* Analyzed gender patterns in speeding violations
* Evaluated gender impact on search decisions
* Calculated average stop duration
* Compared age distribution across violations

---

## 📊 Key Insights

* Behavioral differences observed across genders
* Patterns identified in search rates during stops
* Meaningful insights derived from stop duration analysis
* Age variations identified across violation types
